What This Connector Delivers

A lightweight bridge inside Odoo so the desktop app can pull structured data for reliable ZPL label generation.

  • Export products, barcodes, prices, and quantities in structured formats
  • Generate CSV or JSON payloads for label templates
  • Issue and manage API tokens for desktop app authentication
  • Designed for Odoo Online, Odoo.sh, and on-premise deployments
  • Supports Odoo 14.0 through 19.0

Frequently Asked Questions

Does the connector store data in the cloud?

No. It only exposes data you authorize and the desktop app runs locally on your computer.

Can I export to CSV or JSON?

Yes. Choose the format that matches your label template needs.

Is it compatible with Odoo Online?

Yes. The connector is built for Odoo Online, Odoo.sh, and on-premise installations.

Which printers are supported?

All ZPL-compatible printers including Zebra, TSC, Honeywell, and many others. Over 1800+ label printers supported.

Do I need to pay for the connector?

No. This connector module is free. The desktop app (PynKart ZPL Studio) requires a separate license.
