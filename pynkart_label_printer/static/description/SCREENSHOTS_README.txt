# Screenshots and Banner

For Odoo App Store submission, you need to provide:

## Required Images:

1. **icon.png** (256x256 pixels)
   - App icon shown in Odoo Apps menu
   - Use the icon.svg as a template

2. **banner.png** (1200x400 pixels)
   - Main banner image for app store listing
   - Should showcase the app's main features
   - Include app name and tagline

3. **screenshot_1.png** (1024x768 or larger)
   - Screenshot of label designer interface
   - Show Monaco editor with ZPL code and live preview

4. **screenshot_2.png** (1024x768 or larger)
   - Screenshot of Odoo integration
   - Show field mapping and model selection

5. **screenshot_3.png** (1024x768 or larger)
   - Screenshot of batch printing dashboard
   - Show progress tracking and label preview

## Tools to Create Images:

- **Screenshots**: Windows Snipping Tool (Win + Shift + S)
- **Image Editing**: GIMP (free), Adobe Photoshop, Canva
- **Banner Design**: Canva (free templates available)
- **SVG to PNG**: CloudConvert, Inkscape

## Tips:

- Use consistent branding colors (#00d9a3, #0ea5e9)
- Ensure text is readable
- Show real product UI when possible
- Highlight key features visually
- Use high-resolution images
